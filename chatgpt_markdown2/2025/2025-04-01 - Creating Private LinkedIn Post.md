# Creating Private LinkedIn Post

---
**User:**

How to create a post in LinkedIn without sharing it to others?

---
**Assistant:**

LinkedIn does not have a built-in "private post" feature, but there are a few ways to create a post without sharing it with others:  

### **1. Use the “Only Me” Setting (Limited Option)**  
- Go to **Start a post** on LinkedIn.  
- Click on the **audience selector** (next to your profile picture).  
- If available, choose **Only Me** (Note: This option is mostly for articles, not regular posts).  

### **2. Save as Draft**  
- Start writing a post but don’t publish it.  
- Instead, copy and save the text in **Notes, Google Docs, or Drafts** to review later.  

### **3. Use a Private Account**  
- If you have a secondary LinkedIn account with no connections, post from there to test formatting.  

### **4. Create a Private Group**  
- Create a **LinkedIn Group** with only yourself as a member.  
- Post inside this group so it's not visible to anyone else.
